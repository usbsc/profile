# My Personal Portfolio

A comprehensive personal portfolio website for a freshman at Santa Clara High School in Santa Clara, California.

## 📋 Overview

This portfolio showcases:
- **About Me** - My introduction, interests, and goals
- **School Info** - Santa Clara High School details and programs
- **Bay Area** - Information about Silicon Valley and the Bay Area tech scene
- **UC System** - Overview of the University of California system and colleges
- **Projects** - My projects and achievements, led by my upcoming mobile app Train Me
- **USBSC Studios** - Studio page highlighting Train Me as my main talking point
- **SCUSD News** - Links to Santa Clara Unified School District resources
- **UC News** - Links to UC campus news and admissions information

## 🚀 Features

- **Clean, Minimal Design** - Simple and professional aesthetic
- **Responsive Layout** - Works great on desktop, tablet, and mobile
- **Tab Navigation** - Easy navigation between different sections
- **Keyboard Support** - Use arrow keys to navigate between tabs
- **Fast Performance** - Single HTML file, no external dependencies
- **Dark Theme** - Easy on the eyes with a modern look

## 🎨 Design

- **Color Scheme**: Dark navy background with cyan/blue accents
- **Typography**: Clean sans-serif fonts with good readability
- **Animations**: Smooth transitions and hover effects
- **Mobile-Friendly**: Responsive design for all screen sizes

## 🔧 How to Use

1. **Open the portfolio**: Open `index.html` in any web browser
2. **Navigate tabs**: Click the buttons at the top to switch between sections
3. **Keyboard shortcuts**: Use arrow keys to navigate between tabs
4. **Visit links**: Click on any link to visit external resources

## 📱 Responsive Design

The portfolio is fully responsive and works on:
- ✅ Desktop computers
- ✅ Tablets
- ✅ Mobile phones
- ✅ All modern browsers (Chrome, Firefox, Edge, Safari)

## 🛠️ Customization

To update your portfolio:

1. **Edit personal information**: Open `index.html` and modify content in the "About Me" section
2. **Add new projects**: Add cards to the "Projects & Achievements" section
3. **Update links**: Modify URLs in the "SCUSD News" and "UC News" sections
4. **Change colors**: Modify CSS color values in the `<style>` section
5. **Add sections**: Create new tab-pane divs and corresponding buttons

## 🌐 GitHub Pages

This portfolio is ready for GitHub Pages deployment!

To enable GitHub Pages:
1. Go to your repository settings
2. Scroll to "Pages" section
3. Select "Deploy from a branch"
4. Choose `main` branch and `/` (root) folder
5. Click "Save"

Your site will be live at: `https://usbsc.github.io/profile/`
